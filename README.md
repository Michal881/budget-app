# Budget App 💸

Prosta aplikacja do zarządzania budżetem.

## Funkcje
- dodawanie wydatków
- usuwanie wydatków
- planowanie budżetu
- podsumowanie plan vs wykonanie

## Tech stack
- Backend: FastAPI (Python)
- Frontend: HTML + JavaScript
- Deployment: Render
